# DayFlow HRMS

A Human Resource Management System built with Next.js, TypeScript, Tailwind CSS, and Supabase.

## Prerequisites

- Node.js (version 18 or higher)
- Bun (recommended, version 1.2.15 or higher) or npm

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Odoo_Hackathon_Human_Resource_Management_System
   ```

2. Install dependencies:
   ```bash
   bun install
   ```
   Or if using npm:
   ```bash
   npm install
   ```

## Running the Project

1. Start the development server:
   ```bash
   bun run dev
   ```
   Or if using npm:
   ```bash
   npm run dev
   ```

2. Open your browser and navigate to `http://localhost:3000` to view the application.

## Build for Production

To build the project for production:

```bash
bun run build
```
Or:
```bash
npm run build
```

Then start the production server:

```bash
bun run start
```
Or:
```bash
npm run start
```

## Features

- Employee management
- Attendance tracking
- Leave management
- Payroll processing
- User authentication
- Profile management

## Tech Stack

- **Frontend**: Next.js, React, TypeScript
- **Styling**: Tailwind CSS
- **Backend**: Supabase
- **Authentication**: JWT, bcryptjs
- **Icons**: Lucide React

## Scripts

- `dev`: Start development server with Turbopack
- `build`: Build the application for production
- `start`: Start the production server
- `lint`: Run ESLint for code linting
